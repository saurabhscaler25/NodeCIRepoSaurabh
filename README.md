# Node.js CI

Small Node.js application used as the source project for the Node.js CI lab.

## Contents

```
├── package.json
├── package-lock.json
├── src/
│   └── index.js
├── test/
│   └── index.test.js
├── scripts/
│   └── build.js
└── trigger.txt
```

## Run locally

```bash
npm ci
npm test
npm run build
```
